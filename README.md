# System Monitoring Dashboard

## 📌 Description
The System Monitoring Dashboard is a responsive and interactive web-based dashboard designed to simulate real-time system monitoring. The project visualizes system metrics such as CPU usage, RAM consumption, disk utilization, and network activity through modern charts, indicators, and analytics panels.

The dashboard is inspired by professional monitoring tools used in DevOps, cloud infrastructure, and IT operations environments.

---

## ✨ Features
- Dashboard overview with system metrics
- CPU, RAM, Disk, and Network monitoring
- Interactive performance analytics charts
- Running processes monitoring table
- System alerts and notifications
- Health score monitoring
- Smart recommendations section
- Custom threshold alert management
- Responsive dashboard UI design
- Smooth navigation between modules

## 📊 Dashboard Modules

### Dashboard Overview
Displays overall system performance using radial indicators.

### Performance Analytics
Shows graphical analysis of system metrics using Chart.js.

### Running Processes
Displays simulated process information including CPU and memory usage.

### System Alerts
Provides alert notifications for abnormal system activity.

### Health Score
Displays overall system health analysis.

### Smart Recommendations
Provides optimization suggestions based on system conditions.

### Custom Threshold Alerts
Allows monitoring threshold-based alert conditions.

## 🚀 How to Run
1. Download or clone the repository
2. Open the project folder
3. Run using localhost environment (XAMPP/WAMP)
4. Open the browser and visit localhost
5. Navigate through dashboard modules
